Piscines

Piscines (C/C++/SQL/Java/JS)

Ateliers d'une durée de 1 à 2 semaines, comprenant des horaires de 8h à 24h, pour apprendre un langage de programmation

Piscines

  • Type de projet
    : école
  • Nombre de personnes
    : individuel
  • Période
    : durant les deux semestres ING1 (S5 et S6) - 2022/2023
  • Durées
    : 2 semaines pour le C, 1 semaine pour les autres (C++/Java/JS/SQL)
  • Langages utilisés
    : C, C++, Java, JavaScript, SQL
  • But
    : apprendre un langage de programmation
  • Description
    : immersion totale dans les salles machines de l'Epita, l'intérêt de l'apprentissage en piscine est de faire face aux difficultés du langage appris pendant 16h par jour. Un arbre d'exercices obligatoires devait être validé pendant les 2/3 de la piscine pour avoir la note minimale, puis un rush obligatoire devait être effectué pendant le tiers restant, comprenant un projet individuel, pour mettre en pratique tout ce que nous avions appris.
1 ou 2 semaines
